이미지 관련 태그(img)

        -웹 문서에 사진이나 그림 같은 이미지를 삽입하는 용도의 태그
        -img태그는 내용이 없으므로 종료태그X
        [속성]
        1)src : 삽입할 이미지 파일의 경로, 웹 주소를 지정하는 속성
        2)alt : 이미지를 설명하는 속성
                *이미지가 표시되지 않을 때 대체 텍스트
                *웹 접근성(시각장애인)
        3)width/height : 이미지의 가로/세로 크기 지정하는 속성

src 속성

웹 주소를 이용한 이미지 출력

현재 컴퓨터에 존재하는 이미지 출력 
        -절대 경로 : 하나의 절대적인 기준점을 토대로 목표하는 위치까지 이동하는 경로
        ex) 서울시 중구 남대문로 120 2층 B강의장
        C:\Workspace\4_WEBFRONT\images\cats\cat1.jpg
        
        ex) VSCODE : 워크스페이스(열려있는 최상위 폴더) 기준
        /images/cats/cat1.jpg

        -상대 경로 : 현재 위치를 기준으로 목표하는 위치까지 이동하는 경로
        ex) 옆반

        ex) 작성중인 파일이 기준
        .. : 상위 폴더로 이동
        / : 하위 폴더, 파일 지정
        ../images/cats/cat2.jpg
