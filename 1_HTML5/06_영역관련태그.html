영역 관련 태그

        
block 형식(성질)
 
        -공간을 수직 분할
        -> 수직으로 되어있는 화면에 가로선을 그어 여러행으로 분할하는 것
        -크기를 지정하는 css속성인 width, height를 사용할 수 있다.

        ex) p/pre(문단 나누기), h1~h6(제목 영역 나누기),
            hr(수평선으로 나누기), div(영역 나누기), form(폼 영역 나누기), table(표 영역 나누기)

        
inline 형식(성질)

        -공간을 수평 분할
        -> 수평으로 되어있는 화면에 세로선을 그어 여러 열(컬럼)로 분할하는 것
        -크기를 지정하는 css속성인 width, height를 사용할 수 없다.

        ex) strong/b(영역 내 글자를 굵게), 
            em/i(영역 내 글자 기울이기),
            mark(글자에 노란 배경), 
            span(한 줄 내에서 열의 영역 나누기), 
            a(링크 영역 나누기), 
            img(이미지 영역 나누기), 
            input(입력 영역 나누기)

block/inline 차이점 1 : 줄 바꿈(영역 분할 방식)

block(수직 분할)

첫번째 영역
두번째 영역
세번째 영역
        block 형식 태그는 한 줄을 모두 차지하기 때문에 뒤에 작성되는 내용은 다음 줄에 출력된다.

inline(수평 분할)

첫번째 영역 두번째 영역 세번째 영역 
        inline 형식 태그는 한 줄 내에서 작성된 내용만큼의 칸(열)을 차지하기 때문에
        뒤에 작성되는 내용은 같은 줄에 출력된다.
        
        *inline형식의 태그 작성 시 html 문서에 엔터를 입력해서 작성하는 경우 출력되는 화면에서
        한 칸 띄어쓰기 된 형태로 출력된다. 
        붙여서 출력하려면 엔터, 띄어쓰기를 입력하지 않고 작성해야 한다.
