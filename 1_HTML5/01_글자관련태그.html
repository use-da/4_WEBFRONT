글자 관련 태그

b 태그 : 단순히 글자를 굵게 표시하는 태그
i 태그 : 글자를 기울여 표시하는 태그
ABC(태그는 중복 적용이 가능하다)


strong 태그 : 중요성, 심각함을 표현하는 강조 태그의
em 태그 : 문장 일부분의 중요성을 강조할 때 사용

mark태그 : 형광펜 효과를 나타내는 태그
u태그 : under line(밑줄)을 표시하는 태그
s태그 : 취소선(잘못됨을 표시)을 표시하는 태그
del태그 : 삭제선(없어진 텍스트)을 표시하는 태그
small태그 : 작은 글자를 표시하는 태그
big태그 : 큰 글자를 표시하는 태그
sup 태그(윗첨자) : x2
sub 태그(아랫첨자) : H2O
abbr(abbrevieation:약어) 태그 : 마우스 오버 시 툴팁 출력
