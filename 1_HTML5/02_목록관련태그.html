목록 관련 태그

    ul(unordered list)태그 : 순서가 없는 목록
    ol(ordered list)태그 : 순서가 있는 목록
    li(list item)태그 : ul,ol태그 내부에 들어가 목록의 항목을 나타냄

ol태그

        ol태그의 type속성
        type="a"  -> 소문자 알파벳
        type="A"  -> 대문자 알파벳
        type="i"  -> 소문자 로마숫자
        type="I"  -> 대문자 로마숫자
        type="1"  -> 숫자
    *start속성 : 목록의 시작 숫자를 지정
    *reversed속성 : 목록의 순서를 역순으로 지정
