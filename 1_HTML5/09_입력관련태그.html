text 관련 input태그

        **text 관련 type의 공통속성**
        - size : 입력창의 크기를 지정
        - maxlength : 입력할 수 있는 최대 글자 수를 지정
        - placeholder : 입력창에 기본으로 표시할 값 지정
                        (text,number,email,tel,url,search)               
        - readonly : 읽기 전용으로 지정
        - disabled : 사용 불가로 지정

다중 선택 관련 input 태그

        1)type="checkbox"
            -여러 값 중 원하는 값을 모두 체크(선택)
        2)type="radio"
            -여거 값 중 원하는 값을 하나만 체크(선택)
        **name 속성**
        -input 태그 모두 사용 가능한 속성
        -radio,checkbox에서 서로 관련된 input태그를 묶음으로 다룰 때
        같은 name속성 값을 작성
        -같은 name을 가진 태그들은 하나의 그룹으로 묶임
        - value : 선택된 값의 실제값
        - checked : 선택된 상태로 지정

label 태그

        -input 태그 제목으로 사용되는 태그
        -사용방법
        1)input태그에 id속성을 작성
        2)원하는 위치에 label태그를 작성 후 for속성에 input태그와 같은 id값을 작성

form 태그

        -내부에 작성된 input 태그의 값을 서버로 전달
        ->이 때 전달되는 값의 형식은 k:v 형태
        key == input태그의 name속성 값
        value == input태그에 작성된 값 또는 value속성 값
        -속성
            1)action : 내부에 작성된 input 값을 어디로 전달할 지 지정하는 속성
                        -> 주소를 작성
            2)method(방식) : 데이터를 서버로 전달할 때의 방식을 지정하는 속성
                -get : 전달할 값이 url에 노출
                -post : 전달할 값이 url에 노출되지 않음
            3)enctype : 전달할 데이터의 형식을 지정하는 속성
            4)target : action에 작성된 주소로 이동할 때 새로운 창에서 열릴지 지정하는 속성
                -_self : 현재 창에서 열림
                -_blank : 새로운 창에서 열림
            5)name : form태그의 이름을 지정하는 속성
