CSS선택자 우선순위

        style태그 또는 CSS파일에 작성된 순서대로 해석이 진행되지만 같은 요소에
        여러 CSS속성이 설정되는 경우 우선순위가 적용된다.
        
        * 동일한 우선 순위의 선택자를 이용해 CSS속성을 설정하는 경우
        마지막으로 작성된 CSS속성이 적용된다.(덮어쓰기)
        * 여러 선택자를 이용해 같은 요소에 CSS를 적용하더라도 지정되는 스타일이 
        다른 경우 모두 반영된다.

        1순위 !important; 속성
        2순위 inline-style 속성
        3순위 아이디 선택자
        4순위 클래스 선택자
        5순위 태그 선택자
        ---------------
        6순위 전체 선택자
        7순위 상속
