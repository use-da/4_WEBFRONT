기본 속성 선택자

        특정 속성이 작성된 요소를 선택하는 선택자
        (id,class도 선택은 가능하지만 #, . 기호를 사용하는 것을 권장)

        [작성법]
        선택자[속성명="속성값"]{ CSS 코드;}
        (참고)
        1) 선택자 생략 가능 -> 특정 속성명="속성값"을 가진 모든 요소에 적용
        2) "속성값"에서 쌍따옴표("")는 홑따옴표('')로 대체하거나 생략할 수 있다

        [속성] : 속성만 작성하면 속성이 있는 모든 요소를 선택
        [속성=값] : 속성이 값과 일치하는 요소를 선택
        [속성~=값] : 속성이 값과 일치하는 요소를 선택
        [속성|=값] : 속성이 값과 일치하는 요소를 선택
        [속성^=값] : 속성이 값으로 시작하는 요소를 선택
        [속성$=값] : 속성이 값으로 끝나는 요소를 선택
        [속성*=값] : 속성이 값을 포함하는 요소를 선택자

자식 선택자(>)

        지정된 요소 바로 하위에 존재하는 요소를 선택하는 선택자
        (자손 선택자와 다르게 바로 하위(1단계)에 존재하는 요소만 선택)

        [작성법]
        부모 선택자 > 자식 선택자{ CSS 코드;}
        -> 부모 선택자 하위에 존재하는 자식 선택자 요소를 모두 선택

        [부모 선택자] : 부모 요소를 선택하는 선택자
        [자식 선택자] : 자식 요소를 선택하는 선택자

반응 선택자

        사용자의 움직임에 반응하여 선택하는 선택자
        (마우스, 키보드, 터치 등의 이벤트에 반응하여 선택)

        -선택자:active  -> 선택된 요소를 클릭한 상태
        -선택자:hover   -> 선택된 요소에 마우스를 올리고 내릴 경우
                        1) 마우스를 올리는 경우 : hover에 작성된 CSS코드가 적용
                        2) 마우스를 내리는 경우 : 이전 상태로 돌아감
        -선택자:focus  -> 선택된 요소에 포커스가 갈 경우

상태 선택자

        입력 양식 관련 태그(input, input 관련 태그) 상태에 따라 선택하는 선택자
        (선택된 요소가 활성화 되었는지, 비활성화 되었는지, 체크 되었는지 등)

        :focus  -> 요소에 초점이 맞춰진 경우 
        :checked  -> 요소가 체크 되었을 경우(radio, checkbox)
        :enabled  -> 요소가 활성화(사용 가능) 되었을 경우
        :disabled -> 요소가 비활성화(사용 불가능) 되었을 경우

동위 선택자

        동위 관계(형제 관계)에서 다음에 있는 요소를 선택하는 선택자
        (형제 관계 : 같은 부모를 가지고 있는 요소)

        A
        B (1)
        C (2)

        1) A를 기준으로 바로 다음에 존재하는 B요소를 하나(1) 선택 :
        +A+B{ CSS 코드; }
        2) A를 기준으로 바로 다음에 존재하는 모든 B요소(1), (2) 선택 
        ~(Tidle)A~B{ CSS 코드; }
