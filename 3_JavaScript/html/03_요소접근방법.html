DOM(Document Object Model)

        웹 문서(HTML)의 모든 요소를 객체로 표현하는 방법
        -> 문서 내 특정 요소에 접근하는 방법을 제공

DOM을 이용한 요소 접근 방법

        1. id 속성값으로 요소에 접근 : document.getElementById("id속성값");
        
        2. class 속성값으로 요소에 접근 : document.getElementsByClassName("class속성값");
        3. name 속성값으로 요소에 접근 : document.getElementsByName("name속성값");
        4. tag name 속성값으로 요소에 접근 : document.getElementsByTagName("태그 명");
        5. CSS 선택자로 요소에 접근 
        
            1) document.querySelector("CSS 선택자");
                - 단일 요소를 선택하는 방법
                - 선택된 요소가 여러 개면 첫번째 요소만 선택
            2) document.querySelectorAll("CSS 선택자");
                - 선택된 모든 요소를 얻어와 배열 형태로 반환
                    (각각의 index에 선택된 요소가 하나씩 있음)
