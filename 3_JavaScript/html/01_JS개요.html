스크립트(Script) 언어

        기본 프로그램의 동작을 사용자의 요구에 맞게 수행하는 언어

        별도로 컴파일을 진행하지 않고 인터프리터(interpreter)를 이용해 소스코드를 한 줄 씩 읽어 실행
        -> 컴파일에 소요되는 시간이 없음. 대신 프로그램 실행 속도가 느림

자바스크립트(JavaScript)란?

        웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어

        ECMAScript 표준을 준수하는 범용 스크립트 언어

자바스크립트 작성방법

  • 브라우저 콘솔에 직접 작성
  • 태그에 직접 JS코드를 작성(inline)
  • html 내부에 script 태그를 이용해 작성(internal)
  • html 외부에 파일(.js)를 이용해 작성(external)
