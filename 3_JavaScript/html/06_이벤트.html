이벤트(Event)

        이벤트 : 동작, 행위
        -> 브라우저에서 동작, 행위
            click, keydown, keyup, keypress,
            mouseover, mouseout, mousemove, mousewheel, 
            load, unload, resize, scroll, submit, change,
            focus, blur, select, error, abort, reset, contextmenu, ...
            https://www.w3schools.com/jsref/dom_obj_event.asp
        
        이벤트 리스너 : 이벤트 발생을 대기하다가 감지되면 연결된 기능(함수)를 수행
                        ex) onclick, onchange, onkeyup, onsubmit, ...(on이벤트명)
        이벤트 핸들러 : 이벤트 리스너에 연결된 기능(함수) 이벤트가 발생했을 때
                    수행하고자 하는 내용을 작성하는 함수

인라인 이벤트 모델

        요소 내부에 이벤트 리스너, 핸들러를 작성하는 방법

        on이벤트 명 = "함수명()" 형태로 작성

고전 이벤트 모델

        요소가 가지고 있는 이벤트 속성(이벤트 리스너)에 이벤트 핸들러를 직접 연결하는 방법

표준 이벤트 모델

        - W3C에서 권장하는 이벤트 모델
        - 요소가 가지고 있는 이벤트 속성(이벤트 리스너)에 이벤트 핸들러를 연결하는 방법
        - 요소.addEventListener("이벤트명", 함수);
        - 한 요소에 여러 이벤트 핸들러를 설정 할 수 있다.(고전 이벤트 모델의 단점 보완)

배경색이 입력한 값으로 변하는 박스
